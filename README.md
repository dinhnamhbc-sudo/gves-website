# A Dong Group Website

Website chính của Công ty Cổ phần Tập đoàn Á Đông.

## Định hướng thương hiệu

- Thương hiệu chính: Công ty Cổ phần Tập đoàn Á Đông
- English name: A Dong Group Joint Stock Company
- Brand: A DONG GROUP
- Khẩu hiệu: Kiến tạo giá trị - Đồng hành phát triển

GVES Co., Ltd. được trình bày là công ty thành viên chuyên trách lĩnh vực môi trường, không phải thương hiệu chính của website.

## Lĩnh vực hoạt động

- Tư vấn đầu tư
- Tư vấn thiết kế
- Tư vấn môi trường
- Đầu tư xây dựng
- Tư vấn đấu thầu

## Công nghệ

- Frontend: React + Vite
- Styling: CSS trong `src/styles.css`
- Nội dung: `src/data/siteContent.json`, export qua `src/data/siteData.js`
- Backend nội bộ: Node.js + Express, nếu cần dùng API quản trị nội dung

## Lệnh chạy

```bash
npm install
npm run dev
npm run build
```
